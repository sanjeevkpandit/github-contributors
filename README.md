# View the list of contributors of a github project

## Setup instructions

- Clone repo

  ```bash
  git clone git@github.com:sanjeevkpandit/github-contributors.git
  ```

- For development

  ```bash
  yarn start
  # or
  npm start
  ```

- To prepare a build

  ```bash
  yarn build
  # or
  npm run build
  ```

- To deploy under github-io as gh-pages branch

  ```bash
  yarn deploy
  # or
  npm run deploy
  ```
